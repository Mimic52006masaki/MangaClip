# 画面仕様・画面遷移

## 画面一覧
1. ホーム画面（サイト選択）
2. 漫画まとめ速報画面
3. アニメまとめCH画面
4. バックアップ画面
5. 設定画面（※仮定: APIキーや保存先設定）

## 各画面の仕様

### ホーム画面
- 表示内容：
  - サイト選択ラジオボタン
  - 選択サイトに応じた画面遷移ボタン
- 操作：
  - サイト選択 → 対応画面へ遷移

### 漫画まとめ速報画面
- 表示内容：
  - 日付ごとの1日15件テーブル
  - 各行：時間 / オリジナルタイトル / 生成タイトル / URL / コピーボタン / 削除ボタン
  - 日付ごとのURL一括開きボタン
- 操作：
  - 生成タイトルを個別コピー
  - 1件削除
  - 1日単位のURL一括開き

### アニメまとめCH画面
- 表示内容：
  - 取得順の記事一覧
  - タイトル / URL
  - 全件コピーボタン（画面上1つ）
- 操作：
  - 全件コピー（タイトル,URL形式）

### バックアップ画面
- 表示内容：
  - バックアップ実行ボタン
- 操作：
  - 現在のデータを指定先に保存

## 画面遷移
- ホーム → 漫画まとめ速報
- ホーム → アニメまとめCH
- ホーム → バックアップ
- ホーム → 設定
